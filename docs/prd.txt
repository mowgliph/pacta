# PACTA - Documento de Requisitos del Producto (PRD) - Actualizado

## 1. Visión General
PACTA es una aplicación de escritorio desarrollada con Electron que optimiza la gestión de contratos para grandes empresas. 
La aplicación ofrece capacidades offline y una interfaz intuitiva para el manejo de contratos y suplementos.

## 2. Estado Actual del Proyecto

### 2.1 Implementaciones Completadas
- Estructura base de la aplicación con Electron y React
- Sistema de gestión de contratos básico
- Servicios separados para:
  - Gestión de contratos (`contractService`)
  - Gestión de suplementos (`supplementService`)
  - Estadísticas (`statisticsService`)
- Integración con TanStack Query para manejo de estado y caché
- Base de datos SQLite con Prisma como ORM

### 2.2 En Desarrollo
- Sistema de notificaciones
- Modo público/privado
- Mejoras en la accesibilidad (ARIA attributes)
- Optimización de la arquitectura de servicios

## 4. Especificaciones Técnicas

### 4.1 Stack Tecnológico
- Frontend: React + Electron
- Estado: TanStack Query
- Base de datos: SQLite + Prisma
- Servicios:
  - `contractService`
  - `supplementService`
  - `statisticsService`

### 4.2 Arquitectura
- Patrón de Servicios para lógica de negocio
- Hooks personalizados para gestión de estado
- IPC para comunicación entre procesos
- Sistema de caché optimizado
