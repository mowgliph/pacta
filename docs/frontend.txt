Guías de Frontend - PACTA
.1 Frontend Stack 
* Framework: React con Electron para empaquetar como aplicación de escritorio (.exe). 
* Router: Wouter para navegación entre pantallas. 
* Estado Global: Zustand para gestionar el estado de la aplicación.
.2 What UI Libraries to Install 
* Slider: Flicking (para carruseles en estadísticas). 
* Graficas: Recharts (gráficos en dashboard y estadísticas). 
* Drag'n'Drop: DnDKit (para reordenar contratos o tareas). 
* Estilos: TailwindCSS con Shadcn UI para componentes pre-diseńados. 
* Autenticación: Auth.js (integrado con el backend). 
* Tablas: Tanstack Table (gestión de contratos). 
* Notificaciones: Sonner (alertas de vencimientos). 
* Lista Virtual Infinita: Virtua (para listas largas de contratos). 
* Modales: React Aria Modal (accesibilidad en formularios). 
* Animaciones: Framer Motion (transiciones suaves). 
* Formularios: React Hook Form (crear/editar contratos). 
* Data Fetching: Tanstack Query (consultas al backend local).
.3 Color Palette 
* Fondo: Gradiente oscuro (#1E293B a #2D3748) para un look profesional. 
* Elementos Glass Morphism: Fondo translúcido (rgba(255, 255, 255, 0.2)) con desenfoque (backdrop-filter: blur(10px)). 
* Colores Primarios: 
o Azul oscuro (#1E3A8A) para acciones principales (botones). 
o Verde (#34D399) para estados "Activo" o "Completado". 
o Rojo (#F87171) para "Vencido" o alertas. 
o Gris claro (#D1D5DB) para texto secundario.
.4 Fonts 
* Fuente Principal: Inter (sans-serif, moderna y legible). 
* Tamańos: 
o Títulos: 24px (bold). 
o Subtítulos: 18px (medium). 
o Texto General: 16px (regular).
* Accesibilidad: Contraste mínimo de 4.5:1 (WCAG 2.1).
.5 Loading Speed and Performance Optimization 
* Carga Inicial: Usar lazy loading con React (Suspense) para componentes pesados (ej. gráficos). 
* Optimización de Listas: Virtua para listas virtualizadas (evitar renderizado innecesario). 
* Imágenes: Comprimir imágenes (logos, iconos) y usar formatos modernos (WebP). 
* Caching: Tanstack Query para cachear datos del backend local. 
* Minimización de Re-renders: Usar memoización (React.memo, useMemo) en componentes dinámicos. 
* Tiempo de Carga: Objetivo de <2 segundos para pantallas principales.
